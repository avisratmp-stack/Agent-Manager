# Analyze Issue - Agent

Classifies incoming fallout tickets by issue type, root cause category and confidence score. Groups similar DLQ events to reduce redundant classification.

## Domain
Sky fallout management and ticket resolution.

## Integration
Binds to CES 2.X MCPs and observability MCPs for ticket handling, order management, and system monitoring.
