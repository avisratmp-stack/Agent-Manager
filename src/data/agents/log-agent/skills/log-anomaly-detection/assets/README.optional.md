# assets/

Static resources used by this skill — templates, schemas, data files, images.

## Guidelines
- Store configuration templates, lookup tables, JSON schemas
- Keep files small and focused
- Reference from SKILL.md using relative paths: `assets/my-template.json`

## Examples
- `template.json` — response template
- `schema.yaml` — validation schema
- `lookup-table.csv` — static lookup data
