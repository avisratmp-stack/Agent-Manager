# AI System Monitor
## Capabilities
- Collect CPU, memory, disk, network metrics from NetData
- Multi-AI provider support for analysis (OpenAI, Claude, etc.)
- Auto-remediation: restart services, clear disk, scale resources
- Discord webhook alerts for critical events
