# scripts/

Executable code that agents can run when this skill is activated.

## Guidelines
- Scripts should be self-contained or clearly document dependencies
- Include helpful error messages for failure cases
- Handle edge cases gracefully
- Supported languages depend on the agent runtime (common: Python, Bash, JavaScript)

## Examples
- `extract.py` — data extraction script
- `validate.sh` — input validation
- `transform.js` — data transformation
